## Introduction 
* older animals, plasticity only at the synaptic level 
* younger animals, major structural and functional reorganization (circuit reconstruction, neural proliferation..)
* rendering of the reconstructed fiber tracts : MR tractography

* explication DTI :
    * protons alignés avec un champ magnétique B0 -> tournent autour de l'axe B0 à une phase aléatoire
      * signal à somme nulle
    * RF pulse (onde radio) synchronise les protons -> commencent à tourner en phase, fréquence égale (Larmor), dans le plan transverse xy
      * phase = frequence x temps
      * signal FID : Free Induction Decay (décroît rapidement car protons perdent lasynchronisation)
      * détail: La désoxyhémoglobine (sang pauvre en O₂) est paramagnétique → elle perturbe le champ magnétique local et accélère la désynchronisation des protons -> signal BOLD
    * gradient dans une direction x pendant un temps t -> modifie fréquence localement -> déphase les protons
    * diffusion time / keeps phase
    * inverse gradient same time  
        * if no movement, phase returns to 0 for everyone -> strong signal
        * if movement, dephased signal, sum of all protons compensate eachother -> weak signal
    * Sum up : signal decreases = water diffusion
