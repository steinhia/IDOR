## Introduction 
* AgCC : 3/5% among neurodevelopmental disorders
* prevalence 1/4000
* heterogeneous symptoms
* Alzheimer : callosal damage associated with memory impairment (wang2014)
* split brain : inconsistent findings (preserved/diff)
    * not only CC (also hippocamp), epilepsy past...
    * hyp : visual memory pb because visuo-spatial RH memory traces inaccessible to language LH  (Zaidel1990)
    * limited interaction visuo-spatial/language system -> limited richness of encoding
* AgCC : also inconsistent results
    * auditory memory
        * one study suggests a deficit in retrieving, but not encoding (geffer1994)
            * diff in delayed free recall, not in recognition -> CC must play a role
            * one study reverse pattern -> diff use semantic info to recall ?
        * ... diff in encoding ? CC plays a role ?
    * visual memory : also inconsistent and only case studies
* erickson 2014 :
    * first trial intact : preserved working memory and attention
    * normal learning slope : preserved learning capacity
    * lower total quantity learned : diff of integration with other learned informations ?
    * good retention (delayed recall, % of previously learned items recalled) -> no retrieving difficulty
* sauerwein1994 : impaired recall for complex semantic components (unrelated sentences) 
 
## Etude 
* verbal learning
    * worst on immediate and delayed verbal recall and percent recall (retention) for word pairs 
        * same first trial and learning slopes
        * no difference in details in stories 
        * only immediate recalled impaired for recall of thematic information
            * difficulty to organize semantic information
            * rappellent moins d'infos thématiques, mais se les rappellent bien, et utilisent la structure inhérente, pas besoin de la créer

    * suggests overall less efficient in encoding same amount of information for later recall

* visual learning
  * only delayed recall impaired for faces
  * no difference in abstract figures
  * abstract/concrete ? social/x ?
* complete not stronger deficit compared to partial AgCC
 
## Discussion 
* perspective : spécificité des troubles et pour partial AgCC, quelle partie subsiste ?
* explanation of the contribution of the CC to verbal and visual memory :
    * hemispheric specialization : Adequate encoding of isolated words requires the ability to imagine and generate “meaningful” associations between the unrelated words, which would be difficult without interaction between visual and paralinguistic processing systems primarily located in the right hemisphere and more concrete semantic language systems in the left.
        * but left-handed do not perform better -> lateralization cannot explain everything!
    * processing resource limitation : more neural space when complex task
        * diff with faces can also be explained this way : face is a complex material
